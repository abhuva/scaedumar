# Getting Started

Start close.

Read the ground around the player before choosing a longer route. The prototype is about spending time and condition carefully, not rushing across the map.

## First Steps

1. Use **Inspect** to focus on one kind of sign.
2. Use **PF** to trace nearby travel before committing.
3. Watch fatigue, hydration, and nutrition.
4. Gather plants or water when the ground gives enough reason.
5. Rest when fatigue makes the next choice worse.
