# Controls

The current prototype uses compact labels because the interface is still a development build.

## Main Actions

- **PF** traces a nearby travel route before committing movement.
- **Inspect** narrows attention to one layer of signs.
- **G** searches nearby ground for plants.
- **W** searches for water and fills carried containers when possible.
- **HU** hunts by reading local tracks.
- **SC** starts scouting and animal-possession testing.
- **R** rests to reduce fatigue while time still passes.

## Panels

- **? / Wiki** opens field notes.
- **J** opens the journal.
- **Inventory** opens the inventory panel.
- **RD** opens research and development tools.
